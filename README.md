# 🥗 NutriCare – Personalized Diet Planner

NutriCare is a web-based application that generates personalized diet plans based on user inputs such as age, weight, lifestyle, and fitness goals.

---

## 🚀 Features
- User-friendly interface for entering personal details
- Personalized diet recommendations based on AI calculations
- Multi-step interactive input form (5 steps)
- Real-time nutrition plan generation
- Responsive design for all devices
- Beautiful animations and modern UI with glass-morphism effects

---

## 🛠️ Tech Stack
- HTML5
- CSS3 (Advanced animations, gradients, glass-morphism)
- Vanilla JavaScript (No dependencies)

## 📋 How to Use
1. Click "Start Planning" or navigate to the planner section
2. Answer 5 simple questions about yourself:
   - Your age
   - Your gender
   - Your fitness goal (lose weight, build muscle, maintain)
   - Your activity level (sedentary to very active)
   - Any dietary restrictions (optional)
3. Get your personalized nutrition plan with:
   - Daily calorie recommendations
   - Protein intake targets
   - Carbohydrate intake targets
   - Fat intake targets

## 📌 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/26shaun/nutricare.git
   cd nutricare
   ```
2. Open `index.html` in your browser or use a local server:
   ```bash
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```
